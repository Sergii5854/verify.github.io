Implement function verify(text) which verifies whether parentheses within text are correctly nested. You need to consider three kinds: (), [], <> and only these kinds.

Return - bool

Example: 

    verify("---(++++)----") -> 1
    verify("") -> 1
    verify("before ( middle []) after ") -> 1
    verify(") (") -> 0
    verify("<(   >)") -> 0
    verify("(  [  <>  ()  ]  <>  )") -> 1
    verify("   (      [)") -> 0
